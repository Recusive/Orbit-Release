## Auto-Update

Orbit includes built-in auto-update. When a new version is available, you'll see a notification inside the app. Updates are signed and verified for integrity.

## What is Orbit?

Orbit is a desktop code editor with a deeply integrated AI assistant. It combines a modern editing experience with conversational AI that can read, write, and reason about your entire codebase.

**Core features:**
- Conversational AI coding assistant with tool use
- Full file system, terminal, and git integration
- Syntax highlighting and language server support
- Visual component builder
- Embedded browser panel

## Requirements

- **macOS** 10.15 (Catalina) or later
- **Windows** 10 or later
